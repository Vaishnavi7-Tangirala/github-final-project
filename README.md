# Paradise Nursery

## Project Description
Paradise Nursery is a simple React-based web application for browsing and managing plant products. It allows users to view a list of plants and manage a shopping cart.

## Features
- View available plant products
- Add items to cart
- Remove items from cart
- Simple and user-friendly interface

## Technologies Used
- React.js
- JavaScript
- CSS

## Project Structure
- App.jsx: Main application component
- AboutUs.jsx: About page
- ProductList.jsx: Displays products
- CartItem.jsx: Displays cart items
- CartSlice.jsx: Handles cart logic
- App.css: Styling file
